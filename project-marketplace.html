Marketplace Integrated Accounting System

Role: Quality Assurance (QA Engineer)
Platform: Web-Based Application
Industry: E-Commerce / Financial Management

GB Project Overview

The Marketplace Integrated Accounting System is a web-based accounting application designed to manage financial transactions from online marketplaces such as Shopee and Lazada. The system automatically synchronizes sales data, records financial transactions, and generates financial reports to support business accounting processes.

Key Features Tested

  • Marketplace sales data synchronization (Shopee & Lazada)
  • Automatic revenue recording from marketplace transactions
  • Journal entry automation and transaction reconciliation
  • Financial reporting and sales summary dashboards

GB Testing Approach

  • Requirement-based testing for accounting and integration workflows
  • Integration testing for marketplace API synchronization
  • End-to-end testing for sales-to-accounting processes
  • Regression testing after feature updates
