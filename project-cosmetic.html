Cosmetic Membership System

Cosmetic Membership System is a web application designed to manage cosmetic customer memberships by integrating customer and sales transaction data from TikTok Shop and Shopee. The system centralizes omnichannel data, automates loyalty point calculation, manages membership tiers, and provides reporting to improve customer retention and sales performance.

Project Type: Website
Team Size: 1 Project Manager, 1 Backend Developer, 1 Frontend Developer, 1 QA
Role System: Customer, superadmin, admin

Scope & Responsibilities

  • Defined project scope, timeline, milestones, and deliverables
  • Gathered requirements related to e-commerce integration and membership logic
  • Coordinated backend, frontend, and QA teams
  • Managed sprint planning, task allocation, and progress tracking
  • Ensured data synchronization accuracy and consistency
  • Managed risks, change requests, and issue resolution
  • Coordinated UAT, deployment, and system handover

Key Features Delivered

  • Automated transaction import from TikTok Shop and Shopee
  • Customer data matching and deduplication
  • Loyalty points calculation based on consolidated transactions
  • Membership tier management
  • Promotion and reward configuration
  • Web-based membership and omnichannel sales dashboard

Outcome & Impact

  • Unified omnichannel customer and transaction data into a single web platform
  • Improved loyalty point calculation accuracy across sales channels
  • Enabled data-driven promotions based on consolidated purchase behavior
  • Project delivered on time with stable and scalable web architecture
