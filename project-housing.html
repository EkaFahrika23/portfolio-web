ERP System for Housing Management

ERP System for Housing Management is a web application designed to support operational and financial activities in a residential property company. The system focuses on backoffice data management and cashier operations, enabling centralized control of property data, customer information, transactions, and financial records to improve efficiency, accuracy, and transparency.

Project Type: Website
Team Size: 1 Project Manager, 1 Backend Developer, 1 Frontend Developer, 1 QA
Role System: Superadmin, Cashier

Scope & Responsibilities

  • Defined project scope, timeline, milestones, and deliverables
  • Gathered business requirements from stakeholder
  • Designed system flow for backoffice and cashier modules
  • Coordinated backend, frontend, and QA teams
  • Ensured data integrity, access control, and role-based permissions
  • Managed UAT, deployment, and system handover

Key Features Delivered

  • Master data management (housing projects, units, pricing, customers)
  • Sales and installment transaction recording
  • Cashier payment processing and receipt generation
  • Centralized data validation between operational and financial records

Outcome & Impact

  • All housing operational and financial data managed in one unified system
  • Reduced data duplication and manual reconciliation
  • Faster cashier transaction processing and reporting
  • Improved management decision-making through real-time data access
  • System successfully adopted as the main operational platform
